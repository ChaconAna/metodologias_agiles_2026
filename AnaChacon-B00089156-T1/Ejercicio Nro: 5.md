# Ejercicio Nro: 5
## Enunciado
Proponga cinco elementos que ayuden en el desarrollo de software referido a la propiedad de flexibilidad que se requiere
## Resulocion 
• Diseño para el cambio: Diseñar el sistema para que, si algo cambia, no haya que romper todo, sino solo una parte.
• Ocultamiento de la información: Cada parte del sistema muestra solo lo necesario. Así, se puede cambiar por dentro sin afectar a las demás.
• Desarrollo iterativo e incremental: En vez de hacer todo junto, se construye en partes chicas. Esto permite ajustar y mejorar sobre la marcha.
• Pruebas automatizadas: Tests que revisan el sistema ayudan a detectar errores rápido cuando se hacen cambios.
• Uso de patrones de diseño: Hay formas ya probadas de diseñar software que lo hacen más flexible y fácil de modificar.
